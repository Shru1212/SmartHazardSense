HazardSense

Smart IoT-Based Industrial Safety and Security System

Project Overview

HazardSense is an IoT-based industrial safety and security system designed to detect and monitor hazardous conditions in industrial environments such as factories, warehouses, and plants.

The system aims to reduce industrial accidents by enabling early detection, secure data communication, and real-time monitoring.

Problem Addressed

Traditional industrial safety systems often lack real-time monitoring, secure communication, and scalability. Delayed hazard detection and poor alert mechanisms increase the risk of accidents and operational losses.

Proposed Solution

HazardSense integrates IoT sensors with secure communication protocols to continuously monitor environmental conditions and alert authorities when unsafe situations are detected.

Sensors and Components

  • MQ2 Gas Sensor: Detects leakage of harmful or combustible gases
  • DHT11 Sensor: Measures temperature and humidity
  • ESP32: Microcontroller for sensor interfacing and communication
  • PIR Sensor: Detects unauthorized access or intrusion

Secure Communication using MQTT

Sensor data collected by the ESP32 is transmitted to the backend server using the MQTT protocol. MQTT enables lightweight and real-time data communication suitable for IoT systems.

To enhance security, AES encryption is applied before data transmission, ensuring confidentiality and protection against unauthorized access.

System Architecture

  • Sensors collect environmental data
  • ESP32 processes and encrypts sensor readings
  • Encrypted data is published via MQTT
  • Backend server receives and processes data
  • Dashboard displays real-time monitoring and alerts

Key Features

  • Multi-hazard detection (gas, fire, temperature, intrusion)
  • Secure MQTT-based communication
  • Real-time monitoring dashboard
  • Early warning and alert mechanism
  • Scalable and Industry 4.0 aligned design

Conclusion

HazardSense demonstrates the effective integration of IoT, cybersecurity, and real-time monitoring to improve industrial safety and security. The project aligns with modern industrial requirements and provides a scalable foundation for future enhancements.
